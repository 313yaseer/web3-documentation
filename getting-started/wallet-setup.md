# Wallet Setup Guide

Connect your crypto wallet to interact with our platform.

## Supported Wallets

1. **MetaMask** (recommended)
2. **WalletConnect** (for mobile wallets)
3. **Coinbase Wallet**
4. **Ledger** (hardware wallet)

## Step-by-Step: MetaMask Setup

### 1. Install MetaMask

- **Browser Extension**: https://metamask.io/download/
- **Mobile App**: Available on iOS and Android

### 2. Create or Import Wallet

**New Wallet:**
1. Click "Create a Wallet"
2. Set a strong password
3. **CRITICAL**: Write down your 12-word seed phrase
4. Store seed phrase securely (never share it!)

**Existing Wallet:**
1. Click "Import Wallet"
2. Enter your 12-word seed phrase
3. Set a password

### 3. Connect to Our Platform

1. Navigate to our platform
2. Click "Connect Wallet" button
3. Select "MetaMask"
4. Approve the connection request
5. Verify you're on the correct network (Ethereum Mainnet)

### 4. Verify Connection

You should see:
- Your wallet address (0x...)
- Your ETH balance
- Connected status indicator

## Network Settings

**Ethereum Mainnet:**
- Network Name: Ethereum Mainnet
- RPC URL: https://mainnet.infura.io/v3/...
- Chain ID: 1
- Currency: ETH
- Block Explorer: https://etherscan.io

## Troubleshooting

**Wallet won't connect?**
- Refresh the page
- Unlock MetaMask
- Try disconnecting and reconnecting
- Check you're on the correct network

**Wrong network showing?**
- Click network dropdown in MetaMask
- Select "Ethereum Mainnet"
- Refresh the page

## Security Best Practices

1. ✅ Never share your seed phrase
2. ✅ Verify website URL before connecting
3. ✅ Use hardware wallet for large amounts
4. ✅ Keep MetaMask updated
5. ❌ Don't approve suspicious transactions
